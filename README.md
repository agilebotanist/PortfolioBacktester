# Buy and Hold strategy backtester.

Buy and Hold strategy backtester.

Selects a number of random SP500 stocks
Calculates return on investment
Tests results of yearly rebalancing 

# To deploy
```
pip install poetry

poetry install
```
# To run
```
poetry run streamlit run ./app/front_portfolio.py
```
